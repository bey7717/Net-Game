# Net Mobile Game
Net is a logical puzzle game where users have to rotate various squares on a grid so that they join up into a single connected loop with no loops.

## Purpose
The purpose of this app is create a fun logical puzzle game for people of all ages. People that like games like Sudoku, or Minesweeper would enjoy Net.
